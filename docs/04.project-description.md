### Create Filament NavigationGroup = Company Data on top of Accounting

### Company

1. Create Model and Migration for Company with Softdeletes functionality
2. Table column :
    - id BIGINT PRIMARY KEY AUTO_INCREMENT,
    - name VARCHAR(255) UNIQUE,
    - company_type_id FOREIGN KEY to table company_types,
    - address TEXT,
    - country_id FOREIGN KEY to table countries,
    - province_id FOREIGN KEY to table provinces,
    - district_id FOREIGN KEY to table districts,
    - sub_district_id FOREIGN KEY to table sub_districts,
    - village_id FOREIGN KEY to table villages,
    - postal_code VARCHAR(10),
    - phone_no VARCHAR(20) nullable,
    - fax_no VARCHAR(20) nullable,
    - email VARCHAR(20) nullable,
    - website VARCHAR(20) nullable,
    - contact_name VARCHAR(50) nullable,
    - tax_id VARCHAR(50) nullable,
    - bussiness_license_id VARCHAR(50) nullable,
    - created_at TIMESTAMP,
    - updated_at TIMESTAMP,
    - soft_deletes functionality
3. Create fillable on Company Model and also relationship to foreign tables
4. Create Filament resource for Company with simple and soft delete function
