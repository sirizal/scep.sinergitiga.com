# Project Name : Supply Chain Execution Platform (SCEP)

## Technology Stack : Laravel 13 , LiveWire 4, FilamentPHP 5 , TailwindCSS 4 , PostgreSQL

### Create Filament NavigationGroup = Master Data on top of Company Data

### Customer

1. Create Model and Migration for Customer with Softdeletes functionality
2. Table column :
    - id BIGINT PRIMARY KEY AUTO_INCREMENT,
    - code VARCHAR(10) UNIQUE,
    - name VARCHAR(255) UNIQUE,
    - address TEXT,
    - country_id FOREIGN KEY to table countries NULLABLE,
    - province_id FOREIGN KEY to table provinces NULLABLE,
    - district_id FOREIGN KEY to table districts NULLABLE,
    - sub_district_id FOREIGN KEY to table sub_districts NULLABLE,
    - village_id FOREIGN KEY to table villages NULLABLE,
    - postal_code VARCHAR(10) NULLABLE,
    - phone_no VARCHAR(20) nullable,
    - fax_no VARCHAR(20) nullable,
    - email VARCHAR(255) nullable,
    - website VARCHAR(20) nullable,
    - contact_name VARCHAR(50) nullable,
    - payment_term_id FOREIGN KEY to table payment_terms NULLABLE,
    - credit_limit DECIMAL(18,2) DEFAULT 0,
    - tax_id VARCHAR(50) nullable,
    - bussiness_license_id VARCHAR(50) nullable,
    - is_active BOOLEAN DEFAULT true,
    - created_at TIMESTAMP,
    - updated_at TIMESTAMP,
    - soft_deletes functionality
3. Create fillable on Customer Model and also relationship to foreign tables
4. Create Filament resource for Customer with simple and soft delete function
5. Add Import and export FilamentPHP 5 actions on Table column
