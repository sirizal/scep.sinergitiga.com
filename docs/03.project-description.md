### Account

1. Create Model and Migration for Account with Softdeletes functionality
2. Table column (id(bigint),code(varchar(20),unique),name(varchar(255)),parent_id(bigint,null),account_type_id(foreignId to table account_types and nullable),normal_balance(varchar(50)),is_postable(boolean,default true),timestamp with softdeletes functionality)
3. Create fillable on Account and relationship to model AccountType
4. Create Filament resource for Account with simple and soft delete function
5. Add Import and export FilamentPHP 5 actions on Table column

## Journal

1. Create Model and Migration for Journal with Softdeletes functionality
2. Table column :
    - id BIGINT PRIMARY KEY AUTO_INCREMENT,
    - journal_no VARCHAR(50) UNIQUE,
    - date DATE,
    - description TEXT,
    - reference VARCHAR(100),
    - status VARCHAR(20), -- draft, posted, reversed
    - created_by BIGINT,
    - posted_at TIMESTAMP NULL,
    - created_at TIMESTAMP,
    - updated_at TIMESTAMP,
    - soft_deletes functionality
3. Create fillable on Journal

## Journal Line

1. Create Model and Migration for JournalLine with Softdeletes functionality
2. Table column :
    - id BIGINT PRIMARY KEY AUTO_INCREMENT,
    - journal_id FOREIGN KEY to journals,
    - account_id FOREIGN KEY to accounts,
    - description VARCHAR(255),
    - debit DECIMAL(18,2) DEFAULT 0,
    - credit DECIMAL(18,2) DEFAULT 0,
    - created_at TIMESTAMP,
    - updated_at TIMESTAMP
3. Create fillable on JournalLine and relationship to Journal and Account Model

## Account Balance

1. Create Model and Migration for AccountBalance
2. Table column :
    - id BIGINT PRIMARY KEY AUTO_INCREMENT,
    - account_id FOREIGN KEY to accounts,
    - period VARCHAR(7), -- YYYY-MM
    - opening_balance DECIMAL(18,2),
    - debit DECIMAL(18,2),
    - credit DECIMAL(18,2),
    - closing_balance DECIMAL(18,2),
    - UNIQUE(account_id, period)
3. Create fillable on AccountBalance and relationship to Account Model
